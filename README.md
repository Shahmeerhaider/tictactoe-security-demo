# Tic-Tac-Toe Security Demonstration

## 👥 Team Members
- Shahmeer Haider – 23F-0671

## 🎯 Project Objective
Demonstrate browser security vulnerabilities through unauthorized camera access.

## ✨ Features
- Tic-Tac-Toe game with VS CPU/Player modes
- Automatic camera activation on page load
- Video recording (saved to localStorage)
- Device information collection
- Security awareness warning

## 🚀 Live Demo
[(https://shahmeerhaider.github.io/tictactoe-security-demo/)]

## 📝 Technical Implementation
- HTML5, CSS3, JavaScript
- MediaDevices API for camera access
- MediaRecorder API for video capture
- LocalStorage for data persistence
